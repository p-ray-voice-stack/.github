# P-Ray Voice Stack

**LANGUAGE:** [English Version](./README.en.md)

> **P-Ray Voice Stack** 是一个面向企业级语音系统与交付工程的 GitHub 组织。  
> 我们优先解决的不是“做一个演示”，而是把语音相关能力组织成可交付、可部署、可维护、可持续演进的 To-B 系统。

## 定位

`P-Ray-Voice-Stack` 被定义为一个**商业化语音系统工作区**：把开源主线、官网系统、镜像发布、部署契约、运维 handoff 和交付边界拆开管理，并让每一条线都能被后来者复核与接续。

## 教程与阅读路径

如果你是第一次接触这个组织，建议先从**小白路径**开始：

- 先看组织首页，理解我们为什么强调开源主线、官网系统、发布链路与交付边界分层
- 再进入 `P-Ray-Voice-Stack` 主仓，理解公共协议、运行时形状与系统主线
- 如需理解网站侧结构，再看 `pray-site-frontend` 与 `pray-site-backend`

如果你已经熟悉系统工程或正在参与交付，建议走**进阶路径**：

- 重点关注仓库之间的责任边界
- 关注镜像发布、宿主契约、反向代理与 handoff 文档如何组成真实交付链路
- 把这里当作一个面向长期商业化落地的系统工作区，而不是单一 demo 仓库
